Overview

End-to-end hardware project: designed the PCB, wrote embedded firmware for sensor data acquisition, and built a Python dashboard for real-time monitoring and logging. The system reads temperature and pressure data continuously and displays it live.

The Problem

A test setup needed continuous environmental monitoring but no off-the-shelf logger fit the physical constraints or the specific sensors being used. A custom solution was needed.

Approach

Designed a compact 2-layer PCB in KiCad centered around an STM32 microcontroller. Wrote firmware in C to interface with I2C temperature and SPI pressure sensors, pack the data, and transmit over UART. Built a Python dashboard using matplotlib to receive, parse, and plot the data in real time.

Outcome

Fully working data logger — board assembled and tested, firmware stable, dashboard displaying live readings with CSV export. System ran continuously for 72 hours without dropout.
